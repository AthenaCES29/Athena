#############################
#    Instalador Athena      #
#############################

Recomendo executar cada passo em uma nova instância do terminal:

1º Passo) Execute install.sh (sudo ./install.sh)

2º Passo) Executar manualmente no terminal o último bloco de install.sh 

3º Passo) Execute o comando (pgadmin3). Na interface gráfica, adicione o banco de dado "postgres", nome de usuário "postgres" e senha "123".

4º Passo) migrations.sh irá configurar o banco de dados do projeto Athena.

5º Passo) init.sh inicializa o servidor no endereço http://localhost:8000/
